# OPEN Gift フロントエンド

`git@github.com:tosaken1116/nextjs-app-templates.git`から作成されました。

volta使用

node: `v20.10`

pnpm: `v8.7.4`

## 環境構築

1. ローカルにclone

`git clone https://github.com/powaaaaa/challecara2023-frontend.git`

2. 移動

`cd nextjs-app-template`

3. 開発サーバを起動

`pnpm dev`

## 開発

- Storybookの起動

`pnpm storybook`
